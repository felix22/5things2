An Effective Way to Get Things Done On Time

  • At the start of your day, write the 5 most-important things you need to do
  • Start with #1 and
  • Don't stop until you're done (except for breaks)

That's it! Do this and you'll never have an unproductive day again.

Along with the pomodoro technique, this has been the most effective productivity hack I've ever used.
